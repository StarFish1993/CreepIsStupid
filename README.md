# 施工指南
## 任务
- 采集能量
    - role：[gather,builder,worker]
- 搬运资源
    - role:[remover,builder,worker]
- 建造建筑
    - role:[builder,worker]
- 升级controller
    - role:[builder,worker]
## role
- worker
    - [work,carry,move]
    - 早期工作单位，可以进行所有工作,工作能力较低
- builder
    - [work,carry,carry,move]
    - 建造建筑，升级controller，也可以进行采集和搬用工作
- gather
    - [work,work,move]
    - 采集者，采集对应资源
- remover
    - [carry,carry,move,move]
    - 搬运者
